Privacy Policy

Last updated: March 2026

Language Box

This privacy policy applies to the Language Box application developed by Oguzcan Yildirim.

Data Collection

Language Box does not collect, store, or share any personal data such as names, email addresses, or device identifiers.

The app includes an optional writing exercise feature that sends the text you write to a server for AI-powered feedback. This text is processed in real time and is not stored after the response is returned. No personal information is attached to these requests.

Offline & Online Features

The majority of the application works entirely offline. All lesson content, flashcards, and quiz questions are bundled within the app. No internet connection is required for these features.

The writing exercise feedback feature requires an internet connection to send your text for evaluation. This is the only feature that communicates with an external server.

Local Storage

The following data is stored locally on your device only and is never transmitted to any server:

  • Lesson progress
  • Quiz results
  • Flashcard review data (SM-2 algorithm)
  • Study statistics

This data never leaves your device. All local data is deleted when you uninstall the application.

Third-Party Services

The application does not use any analytics tools, advertising networks, or tracking SDKs. The writing feedback feature uses a secure API hosted on Cloudflare Workers, which processes requests in real time without storing any data.

Data Security

All data transmitted to the writing feedback server is encrypted using HTTPS (TLS/SSL).

Children's Privacy

The application does not collect any personal data and can be safely used by all age groups.
